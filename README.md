# Guild Manager — Jeu de gestion de guilde en C++

> Projet de cours

## Objectif

Jeu tour par tour en mode console où le joueur gère une guilde 
d'aventuriers. Il doit recruter, payer et nourrir ses aventuriers, 
puis les envoyer en quête pour gagner des ressources et progresser.

## Fonctionnalités

- **Recrutement** d'aventuriers avec un coût en ressources
- **Gestion** de la nourriture et des salaires de la guilde
- **Quêtes** à accomplir pour gagner des ressources
- **Progression** et évolution au fil des tours

## Technologies

![C++](https://img.shields.io/badge/C++-00599C?style=flat&logo=c%2B%2B&logoColor=white)

- Langage : **C++**
- Interface : **Console**
- Paradigme : **Orienté objet**

## Compiler et lancer

```bash
g++ -o guild_manager main.cpp
./guild_manager
```
